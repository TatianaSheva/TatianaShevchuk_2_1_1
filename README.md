## Инструмент анализа статистики продаж

**Инструмент анализа статистики продаж** создан для автоматизации части задач из отдела продаж.

Что важно:
* Инструмент содержит массив, принимающий целые числа.
* Метод *max* по всему массиву чисел и возвращает самое большое число.

Как работает:
1. В созданный массив *saleStat* в фигурные скобки загружаются данные в виде цифр через запятую:

![foto](https://i.ibb.co/mFj4q85/2022-09-21-23-24-39.png)
2. Запускается программа.
3. В консоль выводится самое большое число иззагруженных:

![foto](https://i.ibb.co/DM6DGcb/2022-09-21-23-24-26.png)
